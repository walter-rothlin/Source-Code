Willkommen auf der Webseite des Kurses "Distributed and Mobile-Systems" der BWI-A22.

Über dieses Projekt

Diese Webapplikation wurde im Rahmen des Leistungsnachweises für das Fach "Distributed and Mobile-Systems" erstellt. Sie demonstriert die Kernfunktionalitäten eines modernen Webframeworks.

Verwendete Technologien

  • Flask - Ein leichtgewichtiges Python Web-Framework
  • Jinja2 - Template-Engine für Python
  • Bootstrap 5 - Responsives Frontend-Framework
  • Session Management - Benutzerverwaltung über Flask-Sessions
  • Formulare - Benutzerinteraktion über HTML-Formulare

Funktionalitäten

Die App bietet folgende Kernfunktionen:

  • Benutzerregistrierung und -anmeldung
  • Passwortverwaltung
  • Kontaktformular
  • Adressverwaltung (nur für angemeldete Benutzer)
  • Responsive Design für verschiedene Geräte
